Card Independence Test Checklist

Test 1: Independence

Each card operates independently
Resizing one card doesn't affect others
Floating one card doesn't affect others
Each card has its own state

Test 2: Resizability (All 4 Sides)

Live Sync Metrics - Resize from right edge
Live Sync Metrics - Resize from left edge
Live Sync Metrics - Resize from top edge
Live Sync Metrics - Resize from bottom edge
Upload Window - Resize from all edges
Storage Window - Resize from all edges

Test 3: Floatability

Live Sync Metrics - Drag header to float
Upload Window - Drag header to float
Storage Window - Drag header to float
Multiple cards can float simultaneously
Click ⧉ button to dock back

Test 4: Closeability

Live Sync Metrics - Click ✕ to close
Upload Window - Click ✕ to close
Storage Window - Click ✕ to close
Closed cards disappear from view
"Closed Windows" panel appears in top-right

Test 5: Restore Functionality

Click closed window name in restore panel
Window reappears in same position
Floating state is restored if it was floating
Size is restored if it was resized
Content/functionality still works after restore
Restore panel disappears when all windows restored

Test 6: Complete Independence

Close all windows, restore one - others stay closed
Resize one window, close it, restore - size maintained
Float one window, close it, restore - floating state maintained
Each card maintains its own independent state

Browser Console Checks

Open browser console (F12) and verify:
✓ "Created independent entity: [Card Name]" for each card
✓ "Card closed (can be restored): [Card Name]" when closing
✓ "Panel restored: [Card Name]" when restoring
✓ No JavaScript errors
